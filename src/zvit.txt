Лабораторна робота 3
Використана бібліотека axios.
Використовується для здійснення HTTP-запитів з браузера або з середовища виконання JavaScript (Node.js). Вона надає зручний інтерфейс для виконання асинхронних запитів і взаємодії з веб-серверами.
Особливості Axios включають підтримку Promises, можливість встановлення interceptors для обробки запитів та відповідей перед їхнім відправленням або після отримання, підтримку відміни запитів і багато іншого.
Ця бібліотека є популярним інструментом для роботи з HTTP-запитами в різних веб-проектах, тому почав її використовувати по рекомендації знайомих.
